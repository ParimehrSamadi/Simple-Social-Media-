A front-end social media simulation built with vanilla JavaScript and Bootstrap 5. This project demonstrates the ability to integrate external APIs, manage client-side state using LocalStorage, and implement complex DOM manipulation.

Features

Dynamic Content Feed: Fetches user and post data from the JSONPlaceholder API.
Randomized Display: Implements a custom Shuffle Algorithm to randomize the order of users and their corresponding posts upon page load.
Advanced Search: A comprehensive search system utilizing Regular Expressions (Regex) to filter both users (by name/username) and posts (by title/body) simultaneously.
Persistence System: A bookmarking feature that saves selected posts to LocalStorage, allowing data to persist across different pages.
User Profiles: Dynamic rendering of user information with a "More" details toggle for extended profile data.
Responsive UI: Fully responsive layout developed with Bootstrap 5 and custom CSS Media Queries for optimal viewing on mobile and desktop.
UX Enhancements: Custom CSS-based loading animations and interactive UI elements (Like/Save toggles).
Tech Stack

Language: JavaScript (ES6+)
Styling: CSS3, Bootstrap 5
API: JSONPlaceholder REST API
Icons: Bootstrap Icons
Project Structure

index.html - Main feed page.
user.html - User directory and profiles.
saved.html - Bookmarked posts gallery.
